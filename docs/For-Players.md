# For Players

Welcome to the world of Ores Core! If you are a player and want to use Ores Core in your game, here is everything you need to know.

## 🚀 How to Install
Installing Ores Core is simple. Follow these steps:
1. **Choose Your Platform:** Ores Core is available for both **Fabric** and **NeoForge**. Make sure you download the version corresponding to the loader you are using.
2. **Download Ores Core:** Download the latest version of Ores Core from **[Modrinth](https://modrinth.com/mod/ores-core)** or **[CurseForge](https://www.curseforge.com/minecraft/mc-mods/ores-core)**.
3. **Download Ores Mods (Highly Recommended):** If you are the one building your mod list, you **must** install the companion mod **Ores Mods**. It handles the unification of materials so you don't end up with five different types of Copper or Lead from various mods.
    - **Download Ores Mods:** [Modrinth](https://modrinth.com/project/ores-mods) | [CurseForge](https://www.curseforge.com/minecraft/mc-mods/ores-mods)
4. **Drop in Mods Folder:** Drag both `.jar` files into your Minecraft `mods` folder and launch the game!

## 💎 What Does It Change?
- **Unified Inventory:** You will no longer have duplicated items. If multiple mods add Tin, you will only see the unified version from Ores Core in your recipes and inventory.
- **Dynamic Content:** Some items only appear if a mod specifically needs them. This keeps the game clean and performance high.
- **Seamless Gameplay:** All your standard machines (smelting, crushing, etc.) from other mods will work perfectly with Ores Core materials.

## ⚙️ Configuration
Players generally don't need to touch the configuration files, as modpack authors or mod creators usually set everything up. However, if you want to tweak something, you can find our in-game config screen (usually accessible via the mod list).

*Happy mining!*
